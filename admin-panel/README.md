A highly responsive, modern, and sleek administrative dashboard built to manage the Bleapp campus ecosystem. It provides administrators with complete control over students, teachers, courses, and live session attendance.

🔗 **Live Demo:** [https://adminble.netlify.app](https://adminble.netlify.app)

---

## ✨ Key Features

*   📊 **Dashboard Overview:** Get real-time statistics on total students, teachers, active courses, and live sessions.
*   📚 **Course Management:** Create, edit, and delete courses. Assign instructors and manage curriculum details.
*   🧑‍🎓 **Student Enrollment:** Easily enroll or remove students from specific courses.
*   📅 **Session & Attendance Tracking:** View deep analytics on student attendance rates, live session status (Active/Closed), and exact timestamp marking.
*   👥 **User Management:** Complete directory of all registered students and teachers with their LMS IDs and UUIDs.
*   📈 **Student Performance Analytics:** Visual progress bars indicating attendance health (Red for danger zone, Emerald for good standing).
*   📥 **Export Capabilities:** Instantly generate and download CSV reports for course attendance.
*   🌓 **Seamless Theming:** Fully integrated Light/Dark mode with a modern glassmorphism aesthetic and sleek slate/indigo color palette.

---

## 🛠️ Tech Stack

*   **Frontend Framework:** React.js (via Vite)
*   **Styling:** Tailwind CSS (Modern Glassmorphism & Pill UI)
*   **Routing:** React Router DOM
*   **Icons:** Lucide React
*   **HTTP Client:** Axios
*   **Notifications/Toasts:** React Hot Toast
*   **Hosting:** Netlify (Frontend) connected to Vercel (Backend)
